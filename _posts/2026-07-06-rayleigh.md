First, we need to import necessary modules. For this, we only need the holy trinity of python scientific libraries `numpy`, `pandas`, and `matplotlib`. 
```python
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
```

Then, we need to write the rayleigh fractionation equation as a function. There are three common operations that people do when doing this exercise, first is calculating `alpha` or fractionation factor, calculating the `rayleigh` effect, and calculating the cummulative product. Technically I could wrap everything in one function, but for the sake of clarity I split the steps into three differente functions. One thing worth emphasizing is that all of the operations are done on isotope ratios. So you are not supposed to use the delta or epsilon values to express the isotopic ratios in this process.

```python
# Rayeligh fractionation
def alpha(R_reactant, R_product):
    return R_product / R_reactant

def rayleigh(R_initial, f, alpha):
    return R_initial * f**(alpha - 1)

def cummulative_product(R_initial, f, alpha):
    with np.errstate(divide='ignore', invalid='ignore'):
        R_cumulative = R_initial * (1 - f**alpha) / (1 - f)
    return R_cumulative
```

Because calculations are done on isotope ratios, I also wrote these helper function to convert ratios to epsilon notations and vice versa.

```python
# Epsilon notation
def epsilon(R_sample, R_standard):
    return (R_sample / R_standard - 1) * 10000

def R_from_epsilon(epsilon_value, R_standard):
    return R_standard * (1 + epsilon_value / 10000)
```

Here is the specific case that we are exploring. The student wants to explore the behavior of Tl isotope under rayleigh fractionation. I am not going to explain the detailed process here. But, if you are interested, be on the look out for the paper that hopefully will come out soon-ish.

```python
R_SRM997 = 2.38714

# Seawater and Mn-oxide parameters
Eps_sw = -2
Eps_mnox = 16
R_sw = R_from_epsilon(Eps_sw, R_SRM997)
R_mnox = R_from_epsilon(Eps_mnox, R_SRM997)

alpha_mnox = alpha(R_sw, R_mnox)
print(f"alpha (Mn-oxide/Seawater): {alpha_mnox:.4f}")
```

    alpha (Mn-oxide/Seawater): 1.0018



```python
N = np.arange(100, 0.1, -0.1) # Percent remaining from 100% to 0.1%

alpha_value = 1.00005 # Phillips et al (2023) Experiment

f = N / 100 # fraction remaining

# Calculate R for both seawater and Mn-oxide
R = rayleigh(R_sw, f, alpha_mnox)
R_MnOx = alpha_mnox * R

R_MnOx_cum = cummulative_product(R_sw, f, alpha_mnox)


Eps = epsilon(R, R_SRM997) # Seawater epsilon values
Eps_MnOx = epsilon(R_MnOx, R_SRM997)    # Mn-oxide epsilon values
Eps_MnOx_cum = epsilon(R_MnOx_cum, R_SRM997) # Cumulative Mn-oxide epsilon values
```

Here we plot the results of the calculations using the Matplotlib module.

```python
# Plotting
plt.plot(N, Eps, label="Seawater")
plt.plot(N, Eps_MnOx, ls='--', label="Mn-Oxide")
plt.plot(N, Eps_MnOx_cum, ls='-.', label="Cumulative Mn-Oxide")

plt.xlim(100, 0)


plt.xlabel('% Remaining')
plt.ylabel(r'$\epsilon^{205}$Tl')
plt.title('Rayleigh Fractionation of Tl Isotopes')
plt.legend()
plt.grid(ls=':', alpha=0.5)

plt.savefig('rayleigh_fractionation.png', dpi=300)
```
